unable to connect

  1. If there is an error "unable to connect":
    • Disconnect the power cable from the PIN pad.
    • Observe if the PIN pad loses power.
    • If yes, proceed to step 2.
  2. Replace Internet Cable:
    • Replace the internet cable connected to the PIN pad.
    • Check if the issue persists.
    • If the problem continues, raise a ticket

BootLoop and Beeping

  1. Disconnect all the cables at the combox of the pinpad in the below order:
    1. Power Supply
    2. Cat6 Patch to Switch Port
    3. USB Connector (Comm box to POS)
    4. Payment Terminal Connection (HDMI Cord)
  2. Reconnect all the cables at the combox of the pinpad in the below order:
    1. Power Supply
    2. Cat6 Patch to Switch Port
    3. USB Connector (Comm box to POS)
    4. Payment Terminal Connection (HDMI Cord)
